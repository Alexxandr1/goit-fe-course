Задание №1

Блочные теги:

h1...h6 - Эта группа тегов определяет текстовые заголовки разного уровня, которые показывают относительную важность секции, расположенной после заголовка.

p - определяет параграф текста

div - Тег div относится к универсальным блочным контейнерам и применяется в тех случаях, где нужны блочные элементы без дополнительных свойств. В большинстве ситуаций используется как блок-обертка для логически связанного контента. В примере, div оборачивает заголовок и текст, таким образом выделяя статью.

ol - устанавливает нумерованный список, т.е. каждый элемент списка начинается с числа или буквы и увеличивается по нарастающей. ul - устанавливает маркированный список, каждый элемент которого начинается с небольшого символа — маркера. li - определяет отдельный элемент списка.

Строчные теги:

a - Тег a является одним из важных элементов HTML и предназначен для создания ссылок. Путь ссылки определяется в атрибуте href. Ссылка может быть абсолютной, например, https://www.google.com, или относитльной, то есть относительно текущего документа.

span- Универсальный тег, предназначенный для определения строчного элемента внутри документа.

img - Тег img предназначен для отображения на веб-странице изображений в различных графических форматах. Атрибут src указывает браузеру, где можно найти изображение
